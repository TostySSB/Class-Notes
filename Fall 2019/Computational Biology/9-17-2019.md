## Global alignment
---
- Input:
  - S = a c c g t
  - T = c a g t t
- S(match) = +2
- s(mismatch) = 0
- s(insertion/deletion) = -1
- v(i,j) = OPT score of aligning $S[1...i]$ with $T[1...j]$
  - $0\leq i \leq |S|$
  - $0\leq j \leq |T|$
- v(0,0) = 0
- V(i, 0) = $\sum{k = 1}^{i} \delta (S[i], -) = \sum{k = 1}^{i} -1 = -i$

- V(i,j) = max (
    - $v(i-1,j-1) + S(S[i],T[j])$ (mis)match
    - $v(i-1,j) + S(S[i],-)$ deletion
    - $v(i,j-1) + S(-,T[j])$ insertion
  
    )
